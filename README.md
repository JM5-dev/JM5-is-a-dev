### 🎯 Installation

```bash

git clone https://jm5-dev.github.io/JM5-is-a-dev/

# 2. Naviga nella cartella
```

## 🎨 Customization Guide

### 🔄 Come Personalizzare

1. **👤 Informazioni Personali**
   ```html
   <!-- Modifica in index.html -->
   <h1 class="name">Il Tuo Nome</h1>
   <p class="title">La Tua Professione</p>
   ```

2. **📧 Contatti**
   ```html
   <!-- Aggiorna i contatti nella sezione contact-methods -->
   <a href="mailto:tuaemail@example.com">tuaemail@example.com</a>
   ```

3. **🎨 Colori e Tema**
   ```css
   /* Modifica le variabili CSS in style.css */
   :root {
     --orange-yellow-crayola: hsl(0, 100%, 72%); /* Colore principale */
     --eerie-black-2: hsl(240, 2%, 12%);         /* Sfondo scuro */
   }
   ```

4. **📂 Progetti**
   - Sostituisci le immagini in `assets/images/`
   - Aggiorna i contenuti in `index.html`
